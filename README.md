# Smart Calculator (Python Project)

A structured Python calculator project built to practice modular programming, functions, and file handling.

---

## 📌 Features

- Basic arithmetic operations:
  - Addition
  - Subtraction
  - Multiplication
  - Division (with zero-check)
- Number analysis:
  - Positive / Negative / Zero
  - Even / Odd checker
- History system:
  - Saves calculations
  - Loads previous history on startup
  - Clear history option
- Modular structure (separate files for logic)

---

## 🧠 What I Learned

- Python functions
- Module-based project structure
- File handling (read/write)
- Loops and conditionals
- Basic software design thinking
